    # 指令大全

    在這裡可以查閱一些指令用法
    但最新指令不會自動更新
    指令表依機器人的 cr!cmd 為主

    [] 為必填 | {} 為選填
    [@member] 為提及用戶
    [#channel] 為提及頻道

    # admin 管理指令

    指令 說明 用法 備註
    clear 清除指定數量的訊息 cr!clear [Number]
    kick 踢出某位成員 cr!kick [@member]
    ban 封鎖某位成員 cr!ban [@member]
    vote 發出投票 cr!vote [text]
    invites 查閱邀請多少人 cr!invites
    emojiadd 新增表情 cr!emojiadd [:emoji: OR image URL]

    # user 用戶指令

    指令 說明 用法 備註
    setupuser 生成用戶資料 cr!setupuser
    money 查看金錢 cr!money
    language 修改語言 cr!language ["zh_TW"/"zh_CN"/"en_US","ja_JP"] 繁體中文/簡體中文/英文/日文
    marry 結婚指令 cr!marry [@member]
    divorce 離婚指令 cr!divorce
    pet 寵物指令 cr!pet
    card 用戶的資料庫資料 cr!card
    adv 成就表 cr!adv

    # guild 群組指令

    指令 說明 用法 備註
    setup 生成伺服器資料 cr!setup
    ind 新增動態文字頻道 cr!ind 需在主動態文字頻道使用
    clo 關閉動態文字頻道 cr!clo 需在個人頻道使用
    set 編輯動態文字頻道 cr!set [set/name/self/open/nsfw] 同上
    thcreate 建立討論串 cr!thcreate 需在主動態文字頻道使用
    thclose 關閉討論串 cr!thclose 需在個人討論串使用
    thsave 存檔討論串 cr!thsave 同上
    code 智乃兌換代碼 cr!code [code]
    snipes 查閱前 20 個刪除訊息 cr!snipes
    snipe 查閱刪除訊息 cr!snipe {Number} 若沒填數字則顯示第一個
    server 伺服器的資料庫資料 cr!server
    banlist 伺服器的封鎖列表 cr!banlist
    perm 伺服器的權限 cr!perm {@member}

    # game 遊戲指令

    指令 說明 用法 備註
    fortune 查看運勢 cr!fortune 隨機的請勿當真
    together Discord 的 together 功能 cr!together [type]
    mc-user 查看 Minecraft 用戶資料 cr!mc-user [User name]
    mc-hypixel 查看 Hypixel 用戶資料 cr!mc-hypixel [User name]
    mc-server 查看 Minecraft 伺服器資料 cr!mc-server [Server IP OR domain]
    rps 猜拳遊戲(舊版) cr!rps [r/p/s]
    rps2 猜拳遊戲(新版) cr!rps2 [$]
    guess 猜門遊戲 cr!guess
    math 擲骰子 cr!math {Number} 1~Number OR 6
    count 算數 cr!count [算式] ×是* ÷是/ 次方是**
    slot 拉霸機 cr!slot [$]
    love OO 愛你的機率 cr!love {@member} 別當真 對方不是真的愛你
    gay OO 是 GAY 的比例 cr!gay {@member} 別當真 這不是真的
    pick 是或不是 cr!pick [問題] 請問只有是或否的問題
    8ball 8 號球 cr!8ball [問題] 同上
    osucard osu 遊玩圖卡 cr!osucard [Player Name]
    osu osu 遊玩資料 cr!osu [Player Name]

    # normal 普通指令

    指令 說明 用法 備註
    emoji 查閱表情的圖片檔 cr!emoji [:emoji:] 偷圖神奇(喔沒
    help 查閱幫助頁面 cr!help
    oldcommand 查閱舊版指令頁面 cr!oldcommand
    invite 查閱邀請頁面 cr!invite
    version 查閱智乃版本 cr!version
    userinfo 查閱用戶 Discord 資料 cr!userinfo {@member}
    guildinfo 查閱伺服器 Discord 資料 cr!guildinfo
    bi 查閱機器人資料 cr!bi
    gm 查閱伺服器成員狀態表 cr!gm 目前不可用
    role 查閱身分組列表 cr!role
    hi hi? cr!hi
    ping 測試延遲數值 cr!ping
    date 今天日期 cr!date 其實你可以直接看右下角的時間
    avatar 成員頭貼 cr!avatar {@member}
    userbanner 成員橫幅 cr!userbanner {@member} 只有 nitro 用戶才會有橫幅
    savatar / serveravatar 伺服器頭貼 cr!savatar
    banner 伺服器橫幅 cr!banner 只有加成等級 3 的伺服器才有
    say 機器人說話 cr!say [text] 不要拿這指令嚇人
    saychannel 機器人說話 cr!saychannel [#channel] [text] 同上
    tts 機器人說話(tts) cr!tts 同上
    embed 自定義你的 embed cr!embed

    # money 金錢指令

    指令 說明 用法 備註
    work 可以工作賺錢 cr!work [type] 目前只能釣魚 fish
    daily 領取每日金錢 cr!daily 固定有 50 塊 連續簽到可以獲得[5*天數]$
    pay 轉移金錢給別人 cr!pay [@member][$] 會扣 0.2%的手續費,但輸入的金額會全給對方
