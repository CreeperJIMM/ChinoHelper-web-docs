# 使用指令

# 開始使用指令

智乃小幫手的前綴是 cr!

  • 如果要叫出幫助頁面請打 cr!help
  • 如果要叫出指令頁面請打 cr!cmd [type]

提示

如果新版指令頁面看不懂
可以打 cr!oldcmd 來查閱指令表 或是 cr!oldcmd all 查閱完整的指令表
但很多新指令就不會出現在上面喔!

cmd 後面的參數為指令類型
有以下類型:

點擊查閱
  • admin 管理指令
  • user 用戶指令
  • guild 群組指令
  • game 遊戲指令
  • normal 普通指令
  • image 觸圖指令
  • money 金錢指令
  • rank 等級指令
  • text 文字指令
  • other 其他指令

輸入某的類別指令後
就可以看到該類別所屬的指令了
如果要查閱詳細指令說明請打 cr![command] help
例如 cr!chino help

如果使用到需要資料庫的指令(例如金錢或等級指令)
第一次使用指令可能會出現 ❌讀取失敗 的問題
請稍等一會後再使用看看
若還是無法使用請查閱 創建用戶資料
