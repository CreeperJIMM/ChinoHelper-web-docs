# 動態文字頻道

注意!

此功能逐漸淘汰中
可以的話請使用 動態討論串

使用 cr!setup text 創建一個主要類別與主頻道

提示

你可以更改類別或主頻道的名稱
但不可刪除!
若刪除需重新生成一個新的

注意!

若已經有一個動態文字
如果重新打此指令將會覆蓋這個設定
請記得將舊的動態文字頻道刪除!

在已創建的類別動態文字頻道中的 新增頻道
使用 cr!ind 即可創建你的專屬頻道

# 自訂義你的頻道

在你的頻道中可以使用下列指令:

  • cr!set name [你想取的名稱] 即可更改你頻道的名稱

  • cr!set self 或 cr!set open 可以將你的頻道切換成私人或公開頻道

注意!

若成員擁有 管理者 的權限
他還是能看到私人頻道喔!

  • cr!set nsfw 就...你懂的(切換成限制級頻道)
  • cr!clo 刪除你的頻道

提示

成員所創建的頻道的權限裡都會給他管理頻道權限
所以成員可以自行編輯自己的頻道
當然管理員也可以在某成員的頻道使用上述指令

# 動態討論串頻道 Beta

TIP

此功能與動態文字共用動態文字頻道

在動態文字頻道內
使用 cr!thcreate 創建個人討論串 可使用以下指令:

  • cr!thsave 將討論串存檔

提示

你隨時都可以重新打開討論串

  • cr!thclose 刪除討論串

# 動態語音頻道

使用 cr!setup voice 創建一個主要類別與主頻道

提示

你可以更改類別或主頻道的名稱
但不可刪除!
若刪除需重新生成一個新的

注意!

若已經有一個動態語音
如果重新打此指令將會覆蓋這個設定
請記得將舊的動態語音頻道刪除!

加入 新增頻道 這個語音後
會創建一個語音頻道並把你移動過去 斷開語音後就會自動刪除

提示

若你移動到"這群"的其他語音頻道就會把你移動回來!
若移動到"其他群"的語音頻道就會自動刪除頻道

若有其他成員加入你的頻道
你斷開連接不會刪除頻道 需要所有人斷開連接才會刪除頻道

# 訊息偵測器

注意!

此指令違反discord 隱私權相關條目
若使用此指令的請記得告知成員本伺服器使用訊息偵測器
若沒事先警告成員則與機器人無關係!

使用 cr!setup detect <#channel id> 來設定偵測訊息所放置的位置

  • <#channel id> 為你要設的頻道位置
  • 若不填此選項則可以關閉訊息偵測器

提示

請不要將偵測器設在公開頻道!
否則有侵犯隱私權的嫌疑!

警告!

  • 請不要截圖該頻道的任何內容到其他公開場合!
  • 若你有秒發色圖的習慣,請不要在有設定偵測器的群組發
    管理員都看的到!
  • 嚴禁拿成員所刪除的訊息來威脅他人!
    若有此現象,請被害者至智乃官方群組求救
    若核實到有相關嫌疑將會禁止該群組使用偵測器!

# 配置開關

  • cr!setup snipe [true/false] 狙擊訊息 此選項可以設置是否使用狙擊訊息

  • cr!setup safe [true/false] 觸圖功能 此選項可以設置是否可以使用點圖功能(例: cr!chino)

  • cr!setup react [true/false] 訊息反應 此選項可以設置是否要讓機器人回覆你的訊息
    (例: 你打早安 機器人就會回你 早阿)

  • cr!setup slash [true/false] 斜槓指令 此選項可以設置是否可以使用斜槓指令
